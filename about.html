Tentang

SIIMUT merupakan sebuah website Sistem Informasi Meteorologi yang dibuat dan dikembangkan oleh mahasiswa Program Studi Meteorologi Institut Teknologi Bandung.

Pada akhir semester ini, kami diberikan Tugas Besar untuk membuat website pada mata kuliah Sistem Informasi Meteorologi. Dalam pengerjaannya, kami menggunakan beberapa tools diantaranya:

  • HTML & CSS berfungsi untuk membuat website sistem informasi meteorologi SIIMUT
  • Mapinfo 13, digunakan untuk pembuatan peta
  • QGIS 3.16, digunakan untuk pemetaan pada saat praktikum

Tujuan kami membuat website ini adalah untuk memberikan informasi kepada masyarakat terkait pengaruh iklim terhadap hasil produksi kelapa sawit di Sumatera Utara. Harapannya web ini bermanfaat bagi masyarakat terutama para petani yang ada di daerah Sumatera Utara.
